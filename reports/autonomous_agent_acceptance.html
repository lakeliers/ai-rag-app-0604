代码升级点

模块 实现内容 文件
Autonomous Runtime 新增 Goal、Task、AutonomousState 三个核心结构。 autonomous_agent.py
Goal Manager 把用户目标结构化成 objective、deliverable、success_criteria、constraints、assumptions。 autonomous_agent.py
Task Queue 轻量模板拆出收集资料、提取发现、生成交付物三个任务，并记录依赖、状态、预期产物。 autonomous_agent.py
Observe-Act Loop 每轮选择可执行任务，调用现有 agent_runtime.run_agent_pro()，观察结果并更新 State。 autonomous_agent.py
Critic + Reflect 规则判断任务是否有可用答案、资料收集是否有来源；不通过时生成补救任务草案。 autonomous_agent.py
Stop Condition 全局读取 State，判断任务完成、最大步数、连续失败、无可执行任务。 autonomous_agent.py
Human Gate 低风险只读/生成动作自动执行；外发、删除、发布、付款等动作触发确认。 autonomous_agent.py
前端模式 新增“普通问答 / 自主任务”模式，自主任务可查看目标、任务队列、Critic 和 Reflect。 app.py
